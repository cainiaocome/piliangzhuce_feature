# 批量注册

## 特征

* 业务（hostname）

* 用户名
* 头像
* 昵称

* 注册渠道(网页还是app还是第三方，不同注册渠道自动化难度不同)

* 邮箱(可选, 不同邮箱的来源难度不同)

* 手机号码(可选)
* 手机号码号段信息(是否虚拟号段)
* 手机号码归属地
* 手机号码归属地geo
* 手机号码威胁情报(是否在其他平台存在恶意行为)

* 设备指纹

* ip
* ip geo
* 运营商
* ip 时区
* ip asn数据(AS4134, cidr, info)
* ip信誉(非人类访问概率)
* ip是否开放代理

* timestamp

* 注册过程鼠标，键盘记录

* 验证码类型(威胁等级)
* 验证码正确／错误
* 验证码耗时

* 身份证号，姓名(实名相关信息)(可选)

* 密码(可选)
* 密码长度(可选)
* 密码加盐hash结果(可选)

* 该session 30分钟内访问url记录(可选)
